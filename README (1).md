# CNN

Este bloque introduce las Redes Neuronales Convolucionales y su uso en visión por computador. Aquí encontrarás la guía teórica y una práctica guiada para aplicar el modelo a un caso real de clasificación de imágenes.

## 📖 Contenido Disponible

### 📘 Teoría

Guía didáctica con los fundamentos de las CNN: convolución, feature maps, pooling, arquitectura típica, implementación en Keras y casos de uso reales.

➡️ **[Ver guía teórica](./TEORIA.md)**

**Temas cubiertos:**
- Qué es una CNN y cómo piensa
- Cómo funciona paso a paso
- Ejemplo de implementación en Keras
- Casos de uso reales
- Mejores prácticas para visión por computador

### 🎯 Práctica

Ejercicio práctico para entrenar una CNN sobre un problema de clasificación de dígitos con MNIST, con foco en preprocesado, arquitectura, entrenamiento y evaluación.

➡️ **[Ver ejercicio práctico](./PRACTICA.md)**

**Contenido incluido:**
- Preparación de imágenes y normalización
- Diseño de la CNN base
- Entrenamiento y validación
- Comparación de arquitectura base vs mejorada
- Análisis de métricas y sobreajuste

## 🚀 Cómo usar este contenido

1. **Comienza con la teoría** - Revisa la guía teórica para entender cómo aprende una CNN.
2. **Practica lo aprendido** - Completa el caso de MNIST para aplicar la arquitectura en un escenario real.
3. **Consulta cuando lo necesites** - Usa la teoría como referencia durante el entrenamiento y la mejora del modelo.

## 💡 Recomendaciones

- **Empieza por la versión base** del modelo y añade complejidad solo si la validación lo justifica.
- **Cuida el preprocesado**: en visión, normalización y forma del tensor son claves.
- **Compara siempre train y val** para detectar overfitting.
- **Usa el caso MNIST** como plantilla para problemas visuales más complejos.
